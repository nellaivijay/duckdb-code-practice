🏗️ Lakehouse Architecture

Learn modern data lakehouse architecture with DuckDB as the query engine:

Bronze Layer

Raw data ingestion with immutable storage and data lineage tracking

Silver Layer

Cleaned, validated, and standardized data with quality metrics

Gold Layer

Business-ready aggregates optimized for analytics and BI tools

📚 Lab Curriculum

  • Lab 0: Sample Database Setup Generate and load realistic business data
  • Lab 1: Environment Setup Verify installation and explore different interfaces
  • Lab 2: Basic Operations Tables, queries, data types, and CRUD operations
  • Lab 3: Advanced Features Window functions, lakehouse patterns, and complex SQL
  • Lab 4: Python Integration Deep Python API, pandas integration, and ETL pipelines
  • Lab 5: Data Format Operations Parquet, Arrow, and lakehouse format optimization
  • Lab 6: Performance & Optimization Query optimization, indexing, and lakehouse performance
  • Lab 7: Extensions HTTP filesystem, spatial data, and advanced capabilities
  • Lab 8: Real-World Patterns ETL pipelines, data quality, SCD, and production patterns
  • Lab 9: Integration & Production Production deployment, monitoring, and security

🚀 Quick Start

# Clone the repository
git clone https://github.com/nellaivijay/duckdb-code-practice.git
cd duckdb-code-practice

# Install dependencies
pip install -r requirements.txt

# Generate sample data
python3 scripts/generate_sample_data.py
python3 scripts/load_sample_data.py

# Start learning
python3 scripts/setup.py
